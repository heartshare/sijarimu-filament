## Sijarimu v1.5 - DPMPTSP Provinsi Kepulauan Riau

Sijarimu adalah Aplikasi Sistem perizinan online DPMPTSP ini diperuntukkan bagi pemohon yang akan mengajukan permohonan Perizinan Non OSS secara online.

## Tech Stack
- Laravel
- Filament
- Alpine JS
- Tailwind CSS
- MySQL
